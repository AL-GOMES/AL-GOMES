⚙️ Choix du scénario de calcul

Sélectionnez le périmètre des calculs à effectuer. Les étapes affichées dans le bandeau s'adapteront automatiquement.

REMBT → CIBE → TA (complet) — branchement collectif normal NF C 14-100, deux tronçons (C2 puis C1).
REMBT → TA (sans CIBE) — dérivation directe, un seul câble REMBT → tableau abonné, comparé à la limite totale.
CIBE → TA (C1 seul) — seul le tronçon final est calculé ; les étapes amont (CCPI → REMBT, REMBT → CIBE) sont masquées.
